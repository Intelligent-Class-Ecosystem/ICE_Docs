# Intelligent Class Ecosystem 文档

这是 [Intelligent Class Ecosystem](https://github.com/Intelligent-Class-Ecosystem/ICE_Docs) 基于 [VuePress](https://vuejs.press/) 的文档仓库。

## 贡献指南
### 开始编写文档

1. <u>[安装 Node 22](https://nodejs.org/zh-cn/download/)</u> 以上的环境
2. Fork 并打开项目
3. 安装依赖（需要使用 pnpm）
   ```bash
   pnpm install
   ```

4. 启动 VuePress 调试服务器

    ``` bash
    pnpm run docs:dev
    ```

启动 VuePress 调试服务器后，在浏览器中打开终端输出的链接（默认是<u>[http://localhost:8080/](http://localhost:8080/)</u>）即可实时查看文档修改。

本文档使用了 vuepress-plugin-md-enhance
 的一些扩展语法，请尽量直接编辑 Markdown 文本，请不要使用可视化 Markdown 编辑器。推荐使用 [Visual Studio Code](https://code.visualstudio.com/) 编辑文档。

具体编写方法请参阅 [VuePress 文档](https://vuejs.press/) 和 [vuepress-plugin-md-enhance 文档](https://plugin-md-enhance.vuejs.press/zh/)。
